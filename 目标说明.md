# 课程表安卓应用 — 目标说明

## 项目概述
开发一款面向安卓手机的课程表应用，核心定位是**准时提醒 + 语音播报**，解决学生党错过课程、忘记教室的问题。

## 核心功能目标

### 1. 课程管理
- 按周/日/两日视图展示课程表（参考桂电课程表两日视图）
- 支持添加、编辑、删除课程（课程名、教室、教师、起止周、星期几、节次）
- 支持单双周课程区分、跳周、大小学期
- 开学日期设置，自动计算当前是第几周
- **多作息方案切换**：支持夏令时/冬令时等不同作息时间（参考 WakeUp 课程表）
- **上课时间自定义**：每节课的起止时间可自由调整
- **课程备注**：为课程添加重点标记或重要事项备注
- **课程重叠处理**：多课程时间重叠时自动堆叠显示
- **空白格点击**：点击空白时间段快速添加课程

### 2. 上课提醒（语音播报）
- 上课前 N 分钟（可配置，默认 10 分钟）推送通知
- **语音播报内容**："距离 [课程名] 上课还有 10 分钟，地点在 [教室]"
- 支持自定义播报文案
- 课前 5 分钟、1 分钟可设置多级提醒

### 3. 安卓灵动岛适配
- 适配各家厂商的「灵动岛/原子岛/流体云」等实时活动显示：
  - 小米：焦点通知 / 实时状态
  - OPPO：流体云
  - vivo：原子通知
  - 荣耀：灵动胶囊
  - 华为：实况窗
  - 三星：Edge Lighting / 悬浮通知
- 课程开始前显示倒计时、课程名、教室信息
- 点击直达课程详情

### 4. 课程导入（降低录入成本）
- **学校网站自动导入**：内置 WebView 登录教务系统，自动抓取课程表数据
  - 预置主流教务系统解析器（强智、正方、青果等）
  - 支持用户手动粘贴课程表 HTML 源码解析
  - 解析失败时提供可视化字段映射工具
  - **每日自动刷新**：后台定时同步教务系统最新课表（参考桂电课程表）
- **文件导入**：支持 Excel (.xlsx)、CSV、ICS (iCalendar) 格式导入
- **分享码/口令导入**：通过分享码或口令快速导入同学分享的课表（参考 WakeUp 课程表）
- **图片导入**：支持从截图/照片 OCR 识别导入课表（参考 WakeUp 课程表）
- **导出**：支持导出为 Excel / ICS 格式备份或分享

### 5. 日程管理（参考 WakeUp 课程表、桂电课程表）
- **考试安排**：手动添加或从教务系统同步考试信息，支持倒计时显示、特殊标记
- **作业/任务**：添加作业截止日期，支持提醒，按优先级排序
- **自定义事件**：社团活动、自习计划等非课程类日程，与课程统一展示
- **日历视图**：月视图模式，直观查看整月课程与日程分布
- **每日概览**：每天固定时间（可配置，默认 20:00）推送次日课程与待办摘要
- **系统日历同步**：将课程/考试/作业导出到系统日历

### 6. 教务生态扩展（参考桂电课程表）
- **成绩查询**：对接教务系统查询期末成绩、实验成绩、四六级成绩
- **学分绩/GPA 计算**：自动计算加权平均分、绩点
- **空教室查询**：查询指定时间段的空闲教室
- **已选课程查询**：按学期查看已选课程列表
- **免登录跳转**：内置 WebView 快速打开学校教务系统

### 7. 个性化与分享（参考 WakeUp 课程表、Class Widgets）
- **深度外观自定义**：
  - 课程格子颜色、文字颜色/大小自由调整
  - 课表背景图片、格子圆角半径、不透明度可调
  - 课程贴纸/标签装饰
- **主题系统**：浅色/深色/自动跟随系统切换
- **社区主题分享**：上传/下载用户分享的主题配色方案
- **课表分享**：通过分享码、文件、图片方式分享给同学

### 8. 其他功能
- 桌面小部件（Widget）：今日课程一览，支持多种尺寸和样式
- 课程冲突检测
- 多课表切换（不同学期）
- 上课自动静音：检测到上课时间自动切换振动/静音，下课后恢复
- 上下课倒计时：实时显示距离上课/下课还有多久
- 每日自动刷新：后台定时同步教务系统最新数据
- 异常上报：应用崩溃/错误自动上报，便于快速修复问题
- 版本更新检测：自动检测新版本并提示更新

## 非功能目标
- 目标系统：Android 8.0 (API 26) 及以上
- 开发语言：Kotlin
- 架构：MVVM + Jetpack Compose
- 后台保活：使用 AlarmManager + WorkManager 确保提醒可靠触发
- 语音引擎：系统 TTS（Text-to-Speech），无需网络
- 教务系统解析：可扩展的 Parser 插件架构，支持新增学校适配
- 文件解析：Apache POI（Excel）、OpenCSV（CSV）、iCal4j（ICS）
- 日历组件：自定义 CalendarView 或基于 Compose 的日历组件
- 图片 OCR：ML Kit / Tesseract（课表截图识别导入）
- 热更新：支持远程下发配置/解析器更新，无需重新发版
- 异常监控：Firebase Crashlytics 或自建异常上报服务
- 更新检测：远程版本检测 + 增量更新

## 成功标准
- [ ] 提醒准时率 99% 以上（测试覆盖关机重启、Doze 模式、省电模式等场景）
- [ ] 主流品牌手机（小米/OPPO/vivo/荣耀/华为）灵动岛/实况窗正常显示
- [ ] 语音播报在静音/振动模式下仍能正常播放
- [ ] 首次使用 5 分钟内完成第一节课程添加
- [ ] 主流教务系统（强智/正方/青果）导入成功率 90% 以上
- [ ] 文件导入（Excel/CSV/ICS）解析成功率 95% 以上
- [ ] 日程提醒（考试/作业/自定义事件）准时率 99% 以上
- [ ] 上课自动静音触发准确率 99% 以上
